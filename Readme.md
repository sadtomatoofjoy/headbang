Un vieux patch max que j'avais utilisŽ pour faire faire du headbang a une marionette chausette

1-Alimenter les cartes.
2-Ouvrir le patch.
3-Alimenter les servos et le moteur principal.
4-Activer le DSP pour le mouvement Haut/Bas.
5-Bouton STOP pour arrêter. Ne pas clicker sur le bouton start
pour redémarrer le mouvement. Appuyez sur STOP, et ensuite sur
START.
6-Bouton BEGIN THETA pour démarrer la rotation de la tête.
7-Le nombre SPEED définit la vitesse du mouvement. En millisecondes.
